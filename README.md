# CIN3
鉴别CIN3的浸润存在的问题
对于CIN3怀疑早期浸润但HE形态不确定的病例，首选CK/Collagen IV（或CK/Laminin）双重免疫染色来评估基底膜完整性，同时结合多层面连续切片。需要注意的是，CIN3本身即可出现基底膜不连续，因此基底膜缺损并不等同于浸润，仍需结合间质反应（促纤维增生反应）、浸润灶的形态特征等综合判断。对于形态学上类似CIN3累及腺体但深度异常的病例，应警惕CIN3样SCC的可能，此时E-cadherin和Cyclin D1染色可提供额外信息。

以下从形态学线索、间质反应特征和辅助免疫组化标记物三个层面进行系统解读。

## 一、核心问题：为什么基底膜缺损≠浸润？

Richards等的研究通过Laminin免疫过氧化物酶技术系统评估了宫颈基底膜的连续性，发现基底膜小断裂（small breaks）的频率与CIN级别呈正相关——从良性病变的几乎完全连续，到CIN逐渐出现越来越多的小断裂，再到浸润癌时突然转变为碎片化（fragmented）模式。 Stewart等的研究更直接地指出，8例CIN III中有1(https://pubmed.ncbi.nlm.nih.gov/2307415)**5例存在基底膜缺损**，而8例浸润性SCC中有3例在浸润性肿瘤巢周围仍保留部分IV型胶原免疫反应性。 这意味着：2(https://pubmed.ncbi.nlm.nih.gov/1740529)

CIN3中基底膜缺损是常见现象，可能与炎症细胞浸润、基底层增殖活跃导致的上皮-间质界面扰动有关3(https://pubmed.ncbi.nlm.nih.gov/8881920)
浸润癌中基底膜并非完全消失，肿瘤巢周围可重新沉积基底膜成分
因此，单纯依靠基底膜是否连续来判断浸润是不可靠的

## 二、判断真正浸润的形态学线索（HE层面）

al-Nafussi和Hughes的经典研究回顾了120例伴微浸润癌的CIN3，鉴定出与微浸润高度相关的主要特征（出现于>80%微浸润病例，<10%对照CIN3）和次要特征：4(https://pubmed.ncbi.nlm.nih.gov/7962647)

	- 主要特征（高度预测浸润的三联征）：

	表面上皮和深部宫颈隐窝被膨胀性CIN3广泛累及（expansile CIN3）
	腔内坏死（luminal necrosis）——隐窝腔内出现坏死碎屑
	上皮内鳞状成熟（intraepithelial squamous maturation）——即"反常成熟"（paradoxical maturation），在CIN3的全层不成熟细胞中出现局灶性角化或成熟
	在前瞻性验证中，具备全部三项主要特征的36例CIN3中，83%在连续切片或后续活检中发现了微浸润或明确浸润；而缺乏这些特征的64例CIN3中，无一例在连续切片或18个月随访中出现浸润。4(https://pubmed.ncbi.nlm.nih.gov/7962647)

	- 次要特征（提示浸润可能性增加）：

	频繁的核分裂象和凋亡
	隐窝周围同心性纤维增生（pericryptal concentric fibroplasia）——这就是所谓的"促纤维增生反应"
	隐窝周围炎性浸润
	显著的细胞多形性
	核特征改变：明显核仁、染色质透亮
	垂直于基底膜方向排列的深染梭形细胞流（streams of darkly stained spindle cells oriented at right angles to the basement membrane）——这是早期浸润的重要形态学线索

## 三、间质反应（促纤维增生反应）的详细解读

间质反应是判断浸润的关键辅助证据，但需要理解其具体表现形式：

1. 促纤维增生反应（Desmoplastic stromal response）

NCCN宫颈癌病理原则明确指出，SCC的组织学特征为"浸润性和成角的上皮巢，常伴反常成熟，背景为炎症和间质促纤维增生反应"。 这种反应表现为：5(https://www.nccn.org/professionals/physician_gls/pdf/cervical.pdf)

浸润灶周围间质变得致密、纤维化
间质细胞增多，排列紊乱
与周围正常疏松间质形成对比
2. CD34阳性纤维细胞消失 + α-SMA阳性肌纤维母细胞出现

这是间质重塑最具特异性的标志。Barth等的研究发现：6(https://pubmed.ncbi.nlm.nih.gov/12461613)

间质标记	正常宫颈间质	CIN III旁间质	浸润癌间质	参考文献
CD34⁺纤维细胞	密集网络（++++）	密集网络（++++）	几乎完全缺失（-）	1(https://pubmed.ncbi.nlm.nih.gov/2307415), 2(https://pubmed.ncbi.nlm.nih.gov/1740529)
α-SMA⁺肌纤维母细胞	无（-）	6/8例出现（+/++）	大量杂乱排列（++++）	1(https://pubmed.ncbi.nlm.nih.gov/2307415), 2(https://pubmed.ncbi.nlm.nih.gov/1740529)
关键发现：

CD34⁺纤维细胞的局灶性消失对浸润癌具有特异性，是检测小灶浸润的敏感工具6(https://pubmed.ncbi.nlm.nih.gov/12461613)
CIN III旁间质虽然CD34⁺纤维细胞仍保留，但已开始出现α-SMA⁺肌纤维母细胞（6/8例），提示间质已开始发生重塑6(https://pubmed.ncbi.nlm.nih.gov/12461613)
Li等的研究进一步证实，CIN中间质保留弥漫CD34⁺细胞且无α-SMA⁺肌纤维母细胞，而38例SCC中34例间质CD34⁺细胞消失、全部出现α-SMA⁺肌纤维母细胞7(https://pubmed.ncbi.nlm.nih.gov/19383239)
需注意的局限性：慢性炎症也可导致CD34⁺纤维细胞的局灶性减少，因此需结合整体形态学判断。8(https://pubmed.ncbi.nlm.nih.gov/14758552)

3. 新生血管形成

Abulafia等发现微浸润癌的微血管密度显著高于CIS和正常对照（中位数34.5 vs 19 vs 17/HPF），而CIS与正常对照之间无显著差异。间质炎症反应也从正常→CIS→微浸润癌逐步增强。 Leung等的研究同样发现新生血管化在ISCC中占56.5%，而CIN III中为0%。9(https://pubmed.ncbi.nlm.nih.gov/8942829)[[10]](https://pubmed.ncbi.nlm.nih.gov/7512787)

## 四、浸润灶本身的形态学特征（区别于CIN3的线索）

Leung等比较了69例浸润性SCC与48例CIN III，发现以下特征高度提示浸润：10(https://pubmed.ncbi.nlm.nih.gov/7512787)

形态学特征	浸润性SCC	CIN III	P值	参考文献
巨大奇异细胞	66.7%	6.26%	0.01	1(https://pubmed.ncbi.nlm.nih.gov/2307415)
大角化细胞	87%	0%	0.01	1(https://pubmed.ncbi.nlm.nih.gov/2307415)
角化珠	40.6%	0%	0.01	1(https://pubmed.ncbi.nlm.nih.gov/2307415)
坏死	79.7%	8.3%	0.01	1(https://pubmed.ncbi.nlm.nih.gov/2307415)
新生血管化	56.5%	0%	—	1(https://pubmed.ncbi.nlm.nih.gov/2307415)
重要结论：CIN III中不会同时出现两项以上上述特征；若活检中出现两项或以上，即使间质不足以评估浸润，也高度提示浸润性SCC的存在。此外，CIN III中出现大角化细胞或角化珠可能提示宫颈其他部位存在浸润性病变。10(https://pubmed.ncbi.nlm.nih.gov/7512787)

## 五、特殊陷阱：CIN3样鳞癌（CIN3-like SCC）

这是最容易漏诊的浸润模式。Stewart等报道的14例CIN3样SCC中，8例患者在确诊前经历了多次活检，初始活检常被误读为单纯CIN3或仅伴浅表浸润。 其特征为：11(https://pubmed.ncbi.nlm.nih.gov/27681166)

以推挤性边界（pushing borders）深入宫颈壁，形态酷似CIN3累及腺体
无典型的促纤维增生反应或浸润性生长前缘
通过集体细胞侵袭（collective cellular invasion）机制浸润，保留上皮表型（E-cadherin阳性、Cyclin D1阴性），而非经典的上皮-间充质转化11(https://pubmed.ncbi.nlm.nih.gov/27681166)[[12]](https://pubmed.ncbi.nlm.nih.gov/23190089)

## 六、实用诊断策略总结

面对CIN3怀疑浸润的病例，建议采用以下分层策略：

HE形态学评估：首先寻找al-Nafussi三联征（膨胀性隐窝累及、腔内坏死、反常成熟）以及间质反应（促纤维增生、炎性浸润）。若存在，83%的概率伴有浸润，应立即深切片。4(https://pubmed.ncbi.nlm.nih.gov/7962647)

基底膜染色（CK/Collagen IV或CK/Laminin双染）：用于形态学不确定的病例。基底膜完整可较可靠地排除浸润；基底膜不连续需结合间质反应综合判断。13(https://pubmed.ncbi.nlm.nih.gov/15913422)[[2]](https://pubmed.ncbi.nlm.nih.gov/1740529)

间质标记物（CD34 + α-SMA）：CD34⁺纤维细胞的局灶性消失伴α-SMA⁺肌纤维母细胞出现，对早期浸润具有较高特异性，可作为小灶浸润的辅助检测工具。6(https://pubmed.ncbi.nlm.nih.gov/12461613)[[7]](https://pubmed.ncbi.nlm.nih.gov/19383239)

EMT相关标记物（E-cadherin + Cyclin D1）：主要用于鉴别CIN3样SCC与CIN3累及腺体这一特殊诊断陷阱。11(https://pubmed.ncbi.nlm.nih.gov/27681166)[[12]](https://pubmed.ncbi.nlm.nih.gov/23190089)

**综合判断原则**：任何单一标记物都不能独立确定浸润，需将基底膜完整性、间质反应模式、上皮细胞形态学特征三者结合，并始终以HE形态学为基础。
